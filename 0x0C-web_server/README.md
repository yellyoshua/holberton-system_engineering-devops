# 0x0C. Web server

In this project, we will build a web server that will serve static files. We will use the [`http`](https://nodejs.org/api/http.html) module to create a web server. And we will use the [`fs`](https://nodejs.org/api/fs.html) module to read the files from the disk. We will use the [`path`](https://nodejs.org/api/path.html) module to get the path to the file. We will use the [`mime`](https://nodejs.org/api/mime.html) module to get the mime type of the file. We will use the [`url`](https://nodejs.org/api/url.html) module to parse the url.
